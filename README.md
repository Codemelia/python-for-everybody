## git commands
# Initialise repository
```
git init
```

# Clone repository
```
git clone <HTTPS_URL>
```

# Add remote origin
```
git remote add origin <HTTPS_URL>
```

# Pull repository
```
git pull
```

# Check staging status
```
git status
```

# Add changes to staging
```
git add *
```

# Commit changes
```
git commit -m "<COMMIT_MSG>"
```

# Push repository
```
git push --set-upstream origin master
```
